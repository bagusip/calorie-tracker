# CalorieTrack

Aplikasi pelacak kalori statis yang menyimpan data secara lokal di browser. Tidak memerlukan build tool atau server.

## Deploy ke GitHub Pages

1. Buat repository GitHub lalu unggah seluruh file ini.
2. Buka **Settings → Pages** di repository.
3. Pada **Build and deployment**, pilih **Deploy from a branch**.
4. Pilih branch `main`, folder `/(root)`, lalu klik **Save**.
5. GitHub akan menampilkan URL aplikasi setelah deployment selesai.

Data makanan dan target kalori tersimpan hanya di `localStorage` perangkat/browser pengguna.
